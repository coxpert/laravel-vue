# Frontend UI

## Environment

    php v8.1.10
    composer v2.4.4
    node v16.17.0

## Install dependencies

```
composer install
yarn install or npm install
```

## Run

- [Run Backend](https://github.com/coxpert/type-graphql-backend)

- install laravel/valet globally

```
composer global require laravel/vale
```

- Build Laravel Vite

```
yarn dev
```

- Run server

```
php artisan serve
```
